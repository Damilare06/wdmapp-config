EFFIS
*******************************************************************

`EFFIS <https://github.com/wdmapp/effis>`_ is a workflow composition and execution framework, 
which enables efficient code coupling and provides a toolset for visualization and analysis.
It is currently being developed within the ECP Whole Device Modeling project.
